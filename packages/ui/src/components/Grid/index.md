### Usage

```jsx | pure
import { Grid } from 'litecase-ui';
```

<code src="./demo/Grid.tsx"></code>

### API

##### Grid

| Props  | Description |              Type              | Default  |
| :----: | :---------: | :----------------------------: | :------: |
| gutter |  grid gap   | `number` \| `[number, number]` | `[0, 0]` |

##### Grid Item

| Props |   Description   |   Type   | Default |
| :---: | :-------------: | :------: | :-----: |
| span  | grid item width | `number` |   `-`   |
| order | grid item order | `number` |   `0`   |
