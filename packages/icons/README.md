# litecase-ui icons

Some kinds of icons for react.

### Usage

```sh
$ pnpm add @litecase-ui/icons

# or

$ npm i @litecase-ui/icons

```

### Quick Start

```tsx | pure
import { PlusCircle } from '@litecase-ui/icons';

export default () => {
  return (
    <>
      <PlusCircle />
    </>
  );
};
```
